# To Run the Project

- Clone Repo

  ```bash
  git clone https://github.com/SnehalAgrawal/about-react-dummy-firebase-apis.git
  ```

- Jump into the directory

  ```bash
  cd about-react-dummy-firebase-apis
  ```

- Install:

  ```bash
   npm install
  ```

- Run the app:
  ```bash
   npm start
  ```

# APIs and Payload

1. **Notify**\
